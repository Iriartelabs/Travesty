¿Qué es Alpaca?

Alpaca es una API de trading e inversión que proporciona datos de mercado en tiempo real y permite la ejecución de órdenes. Con esta integración podrás:

  • Obtener datos de mercado en tiempo real para tus alertas
  • Monitorear precios actuales de tus activos
  • Acceder a indicadores técnicos y analíticos
  • Verificar automáticamente condiciones para tus alertas

Para utilizar esta integración, necesitas crear una cuenta en Alpaca Markets y obtener tus claves de API.
