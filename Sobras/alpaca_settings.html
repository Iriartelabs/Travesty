¿Dónde obtener las credenciales?

Puedes obtener tus credenciales de API de Alpaca siguiendo estos pasos:

  1. Inicia sesión en Alpaca
  2. Ve a la sección "Paper Account" o "Live Trading"
  3. Haz clic en "Generate New Key"
  4. Copia la API Key y Secret
Consejo: Comienza con Paper Trading para probar sin riesgo antes de usar dinero real.
